# Processed phase-stratified MHQ data (47 items)

- Population: U.S. youth (18–24), Global Mind Project (GMP).
- Variables: 47 MHQ items, common 1–9 scale; higher = worse symptom burden.
- Quality control: removed records with completion time <7 min or >60 min; removed respondents with across-item SD <0.20. No imputation; listwise complete per phase.

## Files
- `phase_Early_Phase_integrated.csv`      (n=518, 47 cols)
- `phase_First_Wave_integrated.csv`       (n=2,321, 47 cols)
- `phase_Second_Wave_integrated.csv`      (n=4,721, 47 cols)
- `phase_Omicron_integrated.csv`          (n=4,023, 47 cols)
- `phase_Post_Omicron_integrated.csv`     (n=2,598, 47 cols)

## Columns
`col_0 ... col_46` correspond to MHQ item IDs 0..46 (see `examples/node_labels.csv` or Supplementary Table S1 for ID → name; domain mapping in `examples/domain_alignment/node2domain.csv`).
